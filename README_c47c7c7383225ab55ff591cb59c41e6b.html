BML

bulletml module for jgame.js,

このモジュールはbulletml.jsのenchant.jsプラグインを参考に、typescriptに修正した上でjgame.jsモジュール対応させたものです。

動作確認用にbulletml.jsも含めてはいますが、最新版を本家から取得して利用される事を推奨します。 https://github.com/daishihmr/bulletml.js
